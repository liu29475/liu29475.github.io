new_keys_object 函数和 new_dict 函数的分析参见 BUILD_MAP，这里不再重复。PyDict_New 函数执行成功会返回一个空字典，失败则返回 NULL。如果返回值是 NULL，则跳转到错误处理。

往下，操作码的参数 oparg 是包含在 {} 中的映射对象的数量。通过 PEEK 宏从值栈中循环地获取这些映射对象，然后调用 PyDict_Update 更新空字典 sum，类似于 sum.update(arg)。函数调用失败时将返回-1，如果调用失败且引发了 AttributeError，则设置 TypeError 说明参数 arg 不合法，然后跳转到错误处理。如果 PyDict_Update 都执行成功，接着将从值栈中弹出这些映射对象，并将更新好的字典压入值栈中供后续操作码使用。
